# 18 — Glossary

- Account/Application User: login/session identity.
- PlayerId: authoritative player identity.
- TeamId: authoritative team identity.
- AssetId: canonical store/gallery asset identity.
- ProductId: store product identity.
- Default Asset: available built-in choice, not owned inventory.
- Owned Asset: asset explicitly acquired/purchased/granted to a player.
- Equipped Asset: owned/default selected for display.
- AppEvents: app-wide synchronization event system.
- Layout Protection: policy preventing unintended UI redesign.
- Phase 2.8: identity/store/inventory hardening phase.
